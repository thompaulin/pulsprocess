# Pulse & Process — site statique

Fichiers principaux :
- index.html
- styles.css
- script.js
- images/

## Modifier les textes
Ouvre `index.html` avec VS Code ou un éditeur de texte.

## Ajouter tes photos
Mets tes images dans le dossier `images/`, puis remplace les placeholders dans `index.html`.

Exemples de noms :
- images/excel-dashboard.jpg
- images/event-sound.jpg
- images/stock-tool.jpg
- images/custom-object.jpg

## Déploiement Vercel
1. Mets ces fichiers dans un dépôt GitHub.
2. Importe le dépôt dans Vercel.
3. Ajoute ton domaine `pulseprocess.fr` dans Settings > Domains.
4. Copie les DNS donnés par Vercel dans IONOS.
